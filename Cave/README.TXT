PROJECT TITLE: Rimuru's adventure
PURPOSE OF PROJECT: Assignature project

HOW TO START THIS PROJECT: Clone repository and starts greenfoot project at greenfoot
AUTHORS: Leonardo López Pérez and Leonardo Puente Esparza
USER INSTRUCTIONS:
For Menu: 
WASD to move, to select a option press ENTER.
For Record:
Press ENTER to view each player and level reached, for continue to Menu press SPACE.
To Play:
WASD to move.
SPACE to shoot slime balls.
Click on frog to show some instructions and start the game on introduction level.
After click the frog, put the name for your slime to start.

Enjoy the game !
